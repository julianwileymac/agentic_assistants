Data Source Management System.

This module provides a handler-based system for managing data source connections
with support for databases, file stores, and APIs.

Inspired by RAGFlow's ingestion pipeline and Dify's data source integration.

Example:
    >>> from agentic_assistants.data.sources import DataSourceManager
    >>> 
    >>> manager = DataSourceManager()
    >>> 
    >>> # Register a PostgreSQL connection
    >>> source = manager.create_source(
    ...     name="my-postgres",
    ...     source_type="database",
    ...     connection_config={
    ...         "driver": "postgresql",
    ...         "host": "localhost",
    ...         "port": 5432,
    ...         "database": "mydb",
    ...         "username": "user",
    ...         "password": "pass",
    ...     }
    ... )
    >>> 
    >>> # Test connection
    >>> result = manager.test_connection(source)
    >>> print(result.success)
