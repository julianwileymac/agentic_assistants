        Hence, by default, encode will raise an error if it encounters text that corresponds

        >>> enc.encode("hello world")

        >>> enc.encode("<|endoftext|>", allowed_special={"<|endoftext|>"})
        [50256]

        [50256]
        >>> enc.encode("<|endoftext|>")
