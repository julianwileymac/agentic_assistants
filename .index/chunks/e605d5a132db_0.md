"""
Data layer convention system.

This module implements a suggested 8-layer data convention inspired by Kedro's
data engineering best practices. It provides helpers for organizing data
across the data pipeline stages.

Layer Structure:
    01_raw        - Immutable raw data (as-is from sources)
    02_intermediate - Cleaned and validated data
    03_primary    - Domain models and business entities
    04_feature    - ML features (feature engineering outputs)
    05_model_input - Training/inference datasets
    06_model_output - Model artifacts and predictions
    07_reporting  - Reports, dashboards, and summaries
    08_external   - Data for external systems

Example:
    >>> from agentic_assistants.data.layers import DataLayerManager, DataLayer
    >>> 
    >>> manager = DataLayerManager(data_root="data")
    >>> 
    >>> # Get layer path
    >>> raw_path = manager.get_layer_path(DataLayer.RAW)
    >>> 
    >>> # List datasets in a layer
    >>> datasets = manager.list_layer_datasets(DataLayer.PRIMARY)
    >>> 
    >>> # Promote dataset between layers
    >>> manager.promote_dataset("users", DataLayer.RAW, DataLayer.INTERMEDIATE)
"""
