# Starter Projects

This directory contains local-first starter projects designed for rapid adaptation.

## Included Starters

- `local-rag-studio`
- `multilevel-storage-kb`
- `document-augmentation-lab`
- `llm-document-augmentation-studio`
- `persistent-memory-assistant`
- `hybrid-cache-rag-assistant`
- `repo-intel-hub`
- `rag-eval-playground`

Each starter includes:

- `README.md` for usage
- `config.yaml` for local configuration
- runnable Python entrypoint script
