# Core Foundation Examples

Small runnable scripts demonstrating `agentic_assistants.core.foundation`.

## Run

```bash
python examples/core/base_models_and_dto.py
python examples/core/repository_service.py
python examples/core/state_machine_and_serialization.py
python examples/core/clients_and_utils.py
```

## Notes

- These examples are local-first and dependency-light.
- Optional integrations (for example MinIO / Prefect) are shown behind safe checks.
