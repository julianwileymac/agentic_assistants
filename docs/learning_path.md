# Learning Path

An opinionated progression for building local-first agentic systems with this repository.

## Beginner

1. Run `agentic config init` and `agentic ollama start`
2. Complete notebooks `01` to `04`
3. Use `agentic index` and `agentic search` on a small local folder
4. Run `agentic templates list` and scaffold `quickstart-chat-agent`

## Intermediate

1. Complete notebooks `05` to `13`
2. Work through cookbook sections:
   - `docs/cookbook/rag`
   - `docs/cookbook/storage`
3. Instantiate and run:
   - `multilevel-storage-kb`
   - `document-augmentation-lab`
4. Explore `examples/starters/repo-intel-hub` for pipeline-driven ingestion

## Advanced

1. Complete notebooks `14` to `25`
2. Build with scoped retrieval and persistent memory:
   - `examples/starters/persistent-memory-assistant`
   - `examples/starters/hybrid-cache-rag-assistant`
3. Implement LLM augmentation and benchmark retrieval:
   - cookbook augmentation recipes
   - `examples/starters/rag-eval-playground`
4. Run server mode and trace synchronization (`agentic traces ...`)
